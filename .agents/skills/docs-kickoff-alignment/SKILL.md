# Kickoff Alignment Skill

## Goal
Keep development decisions aligned with the agreed kickoff.

## Scope anchors
The current phase includes:
- local gameplay on one device
- Flutter mobile app
- freemium model
- premium levels and ads
- optional registration
- Supabase backend base
- Firebase push
- basic analytics
- moderation workflow
- store publication

## Out of scope for this phase
- online multiplayer
- matchmaking with strangers
- complex real-time systems
- large admin platform unless explicitly requested

## When asked to estimate or design
- reduce scope inflation
- favor MVP-first decisions
- call out when a request belongs to phase 2
- prefer deliverables that fit 4 to 6 weeks
