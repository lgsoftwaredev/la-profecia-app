# QA and Release Mobile Skill

## Goal
Prepare stable builds for store submission.

## Validate before release
- guest mode works
- premium gating works
- ads do not appear for premium
- auth providers compile and run
- analytics and push are initialized safely
- app handles offline gameplay
- no crash in main loop of tutorial, setup, round flow, final result

## Release checklist
- Android build
- iOS build
- app icons and splash
- privacy texts
- store metadata placeholders
- purchase products configured
- auth redirect configs checked
